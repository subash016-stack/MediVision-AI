# MediVision AI

## Intelligent Medical Diagnosis and Patient Assistance System

MediVision AI is an Explainable AI-powered web application that assists in medical image analysis, disease prediction, patient risk assessment, and AI-powered medical assistance.

## Technologies

- React.js
- FastAPI
- MongoDB Atlas
- TensorFlow
- Scikit-Learn
- Gemini API
